Hardware Accelerated Body Pose Estimation

Master's thesis project: Blazepose implementation on AMD Versal AI Engines (VCK5000)

Thesis Overview

This project represents my Master's thesis work completed at Northeastern University in 2025. The research explores hardware acceleration of the Blazepose model for real-time human pose estimation using AMD Versal AI Engines, specifically targeting the VCK5000 accelerator card.

The work demonstrates how specialized AI hardware can dramatically improve inference latency and throughput for computer vision workloads, enabling real-time pose estimation in resource-constrained environments.

Technical Approach

System Architecture

  • Blazepose model optimized for AMD AI Engine deployment
  • Custom dataflow architecture leveraging AI Engine tiles
  • High-bandwidth memory interfaces for streaming video data
  • Hardware-software partitioning for optimal performance
  • Real-time video processing pipeline integration

Key Technologies

  • Platform: AMD Versal VCK5000 AI Engine Array
  • Tools: AMD Vitis, Vitis AI, AI Engine Compiler
  • Model Framework: TensorFlow/ONNX for model preparation
  • Languages: C/C++, Python
  • Computer Vision: OpenCV, custom preprocessing pipelines

Research Contributions

  • Performance analysis of pose estimation on AI Engines
  • Optimization strategies for model deployment on heterogeneous architectures
  • Latency and throughput benchmarking against CPU/GPU implementations
  • Design space exploration for resource-accuracy trade-offs

Results & Impact

The thesis successfully demonstrates real-time pose estimation acceleration on AMD Versal AI Engines, achieving significant performance improvements over traditional CPU-based inference while maintaining model accuracy. The work provides insights into deploying complex computer vision models on specialized AI acceleration hardware.

This research contributes to the growing field of edge AI and hardware-accelerated machine learning, with applications in robotics, sports analytics, healthcare monitoring, and human-computer interaction.
